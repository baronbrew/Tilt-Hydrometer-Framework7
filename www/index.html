info TILT Pico is a BLUETOOTH to WIFI adapter based on the RASPBERRY PI PICO W that connects to this app for easy configuration and features remote continuous logging of all your TILT hydrometers without needing another device nearby. It supports cloud logging of multiple sensors at once and transmits the data remotely to this app via GOOGLE SHEETS default cloud service and supports 3rd party cloud services as well. For more information visit: tilthydrometer.com/pages/tilt-pico
